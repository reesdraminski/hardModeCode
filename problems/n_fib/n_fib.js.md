## N-Fibonacci
The Fibonacci sequence is a series of numbers where the next number is found by adding up the two numbers before it.

Create a function that calculates the _n_<sup>th</sup> term of the sequence.

**Function Header:**
```javascript
function n_fib(n) {}
```

**Examples:**
> n-fib(1) => 1
> n-fib(6) => 8
> n-fib(10) => 55
